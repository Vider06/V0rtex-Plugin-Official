# Hall of Fame

The V0RTEX Plugin Hub Hall of Fame recognizes the most active, reliable, and valuable contributors to the plugin ecosystem.

There is no application process. The D.V.V. updates this list periodically based on contribution quality and long-term activity.

---

## Recognition Criteria

| Criterion | Description |
|---|---|
| Quality | Well-written, well-documented, maintainable code |
| Reliability | Clean violation record, accurate changelogs |
| Impact | Useful, widely used, fills real gaps for the security community |
| Consistency | Long-term maintenance, compatibility with new V0RTEX versions |
| Good faith | Respectful communication with D.V.V., accurate submissions |

Quantity alone is not enough. One excellent, maintained plugin outranks ten abandoned ones.

---

## Tiers

### 🥇 Elite Developer
Multiple Elevated plugins. Exceptional, lasting impact on the V0RTEX community. Extremely rare.

### 🥈 Senior Developer
Several Verified plugins. Clean record. Consistent maintenance across multiple V0RTEX versions.

### 🥉 Active Developer
At least one Verified plugin, actively maintained, no open violations.

### ⭐ Notable Contributor
Recognized for a single standout plugin or a specific significant contribution.

## Removal

A developer is removed if they receive a Level 3 or Level 4 enforcement action, abandon all plugins without maintenance, or conduct themselves in a way incompatible with the project. Level 4 removals are permanent.
