# Contributing to the V0RTEX Plugin Hub

This repository is the official hub for Verified, Elevated, and V0RTEX-Made plugins. Contributions are plugins — not code changes to the repository itself.

---

## How to Contribute

### Submitting a new plugin
Read [SUBMITTING.md](SUBMITTING.md) for the full process.

### Submitting a plugin update
Read [UPDATING.md](UPDATING.md) for the full process.

### Requesting Elevated permissions
Your plugin must already be Verified. Read the Elevated section in [SUBMITTING.md](SUBMITTING.md#requesting-elevated-permissions).

### Reporting a plugin
Open a Discussion in **🚨 Report** or use the report form in Discussions.

---

## What You Cannot Contribute

- Changes to repository documentation or infrastructure files — these are maintained exclusively by the D.V.V.
- Plugins on behalf of another developer
- V0RTEX-Made plugins — this class is reserved for Vider_06 only

---

## Review

All submissions are reviewed by the **D.V.V. — Dipartimento di Verifica V0RTEX**. There is no guaranteed review timeline. Do not open duplicate submissions.
