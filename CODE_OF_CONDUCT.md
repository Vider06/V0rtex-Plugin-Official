# Code of Conduct — V0RTEX Plugin Hub

The V0RTEX Plugin Hub is a space for developers building security tooling. These rules apply to all interactions in this repository — Discussions, submissions, and reports.

---

## Expected Behavior

- Be respectful and constructive in all communications
- Accept D.V.V. decisions without harassment
- Submit accurate, honest information in all forms
- Report issues in good faith using the correct channels

## Unacceptable Behavior

- Harassment, abuse, or threats directed at the D.V.V. or other developers
- Submitting plugins or requests in bad faith
- Attempting to circumvent the review or enforcement system
- Impersonating other developers or D.V.V. members
- Spam, duplicate submissions, or deliberate disruption of the review process

## Enforcement

Violations of this Code of Conduct may result in:

- Discussion or comment removal
- Temporary or permanent ban from repository interactions
- Escalation to plugin enforcement (Level 1–4) if the violation is related to a plugin submission

Severe or repeated violations are handled at the D.V.V.'s discretion, up to a permanent account ban consistent with Level 4 enforcement.

## Reporting

If someone is violating this Code of Conduct, open a Discussion in **🚨 Report** or contact [@Vider06](https://github.com/Vider06) directly.
